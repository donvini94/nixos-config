#+TITLE: NixOS Configuration
#+DESCRIPTION: Flake-based NixOS config for a desktop (dracula) and home server (alucard)

* Overview

Flake-based NixOS configuration managing two systems:
- *dracula* — Desktop workstation (AMD CPU, NVIDIA GPU, Hyprland + caelestia-shell)
- *alucard* — Home server (Hetzner OpenStack VM, nginx reverse proxy, Docker media stack)

The flake provides two host-builder functions:
- =mkDesktopHost "hostname"= — Full desktop with Hyprland, home-manager, sops, hosts blocking
- =mkServerHost "hostname"= — Minimal server (host imports what it needs)

For detailed architecture and module documentation, see [[file:CLAUDE.md][CLAUDE.md]].

* Build Commands

#+begin_src bash
# Desktop
sudo nixos-rebuild switch --flake .#dracula

# Server
sudo nixos-rebuild switch --flake .#alucard

# Test build (no switch)
sudo nixos-rebuild build --flake .#dracula

# Update flake inputs
nix flake update

# Check / show flake
nix flake check
nix flake show
#+end_src

* Adding a New Desktop Host

#+begin_src nix
# 1. In flake.nix, add one line:
newhost = mkDesktopHost "newhost";
#+end_src

#+begin_src bash
# 2. Create the host directory:
mkdir -p hosts/newhost
#+end_src

#+begin_src nix
# 3. hosts/newhost/default.nix — import shared modules + set identity:
{ lib, pkgs, username, ... }:
{
  imports = [
    ../../modules/desktop.nix    # Full desktop stack
    ../../modules/nvidia.nix     # Opt-in: only for NVIDIA GPUs
    ../../modules/gaming.nix     # Opt-in: Steam, Wine, etc.
    ./hardware.nix
  ];
  networking.hostName = "newhost";
  users.users."${username}" = {
    isNormalUser = true;
    extraGroups = [ "networkmanager" "wheel" ];
    packages = with pkgs; [ firefox ];
  };
  system.stateVersion = "24.11";
}
#+end_src

#+begin_src bash
# 4. Generate hardware config on the target machine:
nixos-generate-config --show-hardware-config > hosts/newhost/hardware.nix
#+end_src

#+begin_src bash
# 5. Build and switch:
sudo nixos-rebuild switch --flake .#newhost
#+end_src

* Adapting This Config

1. Fork and clone
2. Create your host under =hosts/<name>/= following the pattern above
3. Enable/disable opt-in modules (=nvidia.nix=, =gaming.nix=) as needed
4. For a server: adapt =hosts/alucard/= — each file is self-contained
5. Set up your own sops-nix age keys in =.sops.yaml= and re-encrypt secrets

See [[file:CLAUDE.md][CLAUDE.md]] for detailed architecture notes and module documentation.
